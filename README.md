# HR Analytics SQL Project

## Overview

The **HR Analytics SQL Project** demonstrates the use of SQL to analyze employee data and generate meaningful business insights. The project covers database creation, table design, data insertion, exploratory data analysis (EDA), and business analysis using SQL queries. It focuses on employee demographics, salary distribution, department performance, attendance, training participation, and attrition analysis.

---

## Problem Statement

Analyze HR employee data using SQL to identify workforce trends, salary distribution, employee attrition, attendance, and training participation. The objective is to answer business questions and support data-driven HR decision-making.

---

## Objectives

* Create an HR Analytics database.
* Design relational tables using primary and foreign keys.
* Insert employee, department, attendance, and training data.
* Perform data cleaning and exploratory data analysis (EDA).
* Analyze employee and department performance using SQL.
* Generate business insights through SQL queries.

---

## Database Tables

* **Departments**
* **Employees**

* DDL (CREATE DATABASE, CREATE TABLE)

* INNER JOIN

* Common Table Expressions (CTEs)

## Analysis Performed

* Data exploration and NULL value checking.
* Employee filtering based on age, gender, salary, and hire date.
* Department-wise salary analysis.
* Employee count by gender and department.
* Attendance analysis.
* Training participation analysis.
* Employees without training records.
* Employees earning above the average salary.
* Salary ranking using window functions.
* Reusable SQL views.
* Business analysis using CTEs.

---

## Key Insights

* Identified department-wise salary distribution.
* Analyzed employee demographics and workforce distribution.
* Evaluated attendance and training participation.
* Ranked employees based on salary.
* Retrieved employees earning above the company average salary.
* Generated business insights using joins, subqueries, CTEs, and window functions.
